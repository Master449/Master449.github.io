Why does this resume website look like this?

I know a lot about website design but I wanted this website to be fast, and as minimal as possible. I wanted to replicate parts of the internet that I remember growing up. The site currently weighs in at 47.3 Kb. This website can fit 100x times in an iPhone 15 HEIF photo. I also wanted no ads, analytics, tracking, and finger printing. And any JavaScript is super minimal. This was also a project to test how lean I can make any assets this site needs. Adding onto that, the entire website is loaded currently.
